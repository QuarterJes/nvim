## Installation

This is probably how you get the setup.\
Prerequisites: nvim

```zsh
  mkdir ~/.config/nvim
  cd ~/.config/nvim
  git clone https://github.com/QuarterJes/nvim
```
This is how you install in a MacOS system, probably similar for unix systems.
